## Installation

You can install the package via composer:

```bash
composer require arvi/unit-conversion
```

## Usage

```php
$skeleton = new UnitConversion\UnitConversion();
echo $skeleton->echoPhrase('Hello, UnitConversion!');
```

## Testing

```bash
composer test
```
